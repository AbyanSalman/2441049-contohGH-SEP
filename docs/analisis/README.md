# Branch Analisis

Dokumen pada branch ini berfokus pada kebutuhan awal, metodologi, dan batasan sistem.

## Isi Utama
- Justifikasi pemilihan Agile
- Tahap pengembangan per sprint
- WBS / dekomposisi kerja
- Identifikasi aktor dan hak akses
- Alur proses utama seperti login, kelola transaksi, dan notifikasi stok

## Rangkuman dari Makalah
Makalah menjelaskan bahwa sistem Arcatory mengakomodasi tiga aktor utama: Admin Gudang, Manajer Gudang, dan Operator Gudang. Admin memiliki kendali penuh, Manajer melakukan verifikasi dan pemantauan, sedangkan Operator menangani transaksi harian.
