# Overview Proyek Arcatory

Arcatory adalah sistem inventori gudang berbasis web yang dirancang untuk meningkatkan efisiensi pengelolaan data barang, transaksi stok, monitoring, dan notifikasi. Konsep proyek ini menekankan penggunaan Agile, pembagian tugas melalui WBS, pemodelan UML, dan rancangan antarmuka berbasis peran. fileciteturn0file0

## Tujuan
- Mengurangi pencatatan manual
- Menjaga integritas data stok
- Mempercepat proses operasional gudang
- Memberikan informasi stok secara lebih real-time
