# Arcatory — Sistem Inventori Gudang Berbasis Web

Repository ini disusun berdasarkan proyek **Arcatory** yang membahas sistem inventori gudang berbasis web dengan pendekatan **Agile**, pemodelan **UML**, dan rancangan antarmuka berbasis peran. Sumber konsep utama diambil dari makalah yang menjelaskan metodologi, WBS, aktor sistem, alur activity/sequence, serta desain antarmuka untuk Admin Gudang, Manajer Gudang, dan Operator Gudang.

## Ringkasan Proyek
Arcatory dirancang untuk membantu pengelolaan data gudang secara lebih efisien melalui fitur-fitur berikut:

- manajemen master data
- transaksi barang masuk/keluar
- pemantauan stok
- notifikasi stok kritis
- pelaporan
- hak akses berbasis peran

## Struktur Branch
Repository ini disiapkan dengan 3 branch kerja utama:

- `analisis` → berisi hasil analisis kebutuhan, metodologi, aktor, dan alur sistem
- `desain` → berisi pemodelan sistem, rancangan UML, dan rancangan antarmuka
- `implementasi` → berisi struktur implementasi, komponen aplikasi, dan panduan pengembangan

## Cara Pakai
Setelah repository di-clone, pindah ke branch yang dibutuhkan:

```bash
git checkout analisis
git checkout desain
git checkout implementasi
```

## Catatan
Isi repository ini dibuat sebagai template siap upload ke GitHub agar mudah dipresentasikan sebagai hasil kerja proyek.
